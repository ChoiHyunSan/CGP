폭탄이 사라짐과 동시에 가로세로 + 2칸 만큼의 위치에 불꽃을 생성

불꽃은 같은 위치에 여러 각도로 생성한다.

일정 시간 뒤에 사라짐

불꽃 오브젝트의 콜라이더에 접촉한 벽을 제외한 모든 오브젝트는 제거한다.



0. 적 오브젝트 생성
1. 콜라이더 구현
2. 불꽃 이펙트 구현 (별도의 오브젝트로 생성) -> 구현은 완료, 최적화해서 만들기
 - 빌보드 기능
 - 폭탄 위치값을 받아와서 그 주변 위치에 생성
 - 불 이펙트 인자값들 테스트 해보기 ( 좀더 자연스럽게 만들기 위함)


3. 게임 매니저 구현 ( 게임 정보값 , 게임 진행 상태 저장 및 갱신 )
4. UI 정비



5. 맵 디자인 & 맵 생성
6. 


// 맵 및 Enemy 오브젝트 개발 아이디어
1. 복셀 맵의 좌표값과 해당 복셀의 특성을 각각 저장한다.
-> 해당 정보를 이용하여 각각의 오브젝트에 기능을 부여한다.

1) Enemy : 배열 정보를 이용하여 PathFinder, A* algorithm을 구현해준다.
2) Bomb : 기존 플레이어의 위치값에 생성되는 것을 플레이어의 위치와 가장 가까운 복셀 위치값에 생성한다.




먼저 개발할 것
1. 플레이어 위치를 반올림하여 값을 저장
2. 폭탄 설치 위치를 1번으로 대체
3. 봄버맨 맵 1개를 같은 디자인으로 만들어놓기
4. 맵 정보를 배열로 저장 - 2차원 배열
5. 폭탄이 터지면서 Fire 이펙트를 생성하게 저장
